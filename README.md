# Latihan Soal DSA

Setiap soal berada di file `main.cpp` masing-masing folder dan penjelasan soalnya ditulis sebagai komentar di dalam file tersebut.

Struktur folder:

- `soal-01-array-pointer/main.cpp`
- `soal-02-big-o-linear-search/main.cpp`
- `soal-03-struct-arrow-operator/main.cpp`
- `soal-04-stack-overflow/main.cpp`
- `soal-05-queue-underflow/main.cpp`
- `soal-06-singly-linked-list-add-last/main.cpp`
- `soal-07-double-linked-list-delete-first/main.cpp`
- `soal-08-jump-search/main.cpp`
- `soal-09-insertion-sort/main.cpp`
- `soal-10-bst-inorder/main.cpp`

Contoh compile:

```bash
g++ soal-01-array-pointer/main.cpp -o soal1
./soal1
```
