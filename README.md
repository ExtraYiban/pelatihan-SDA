# Latihan Soal DSA (1-50)

Repositori ini berisi 50 latihan Data Structures and Algorithms (DSA) berbasis C++.
Setiap soal berada di folder terpisah dan file utama latihan ada di `main.cpp`.

Format di setiap soal:

- Judul dan deskripsi materi
- Konsep kunci
- Dokumentasi fungsi (`@brief`, `@param`, `@logic`)
- Blok TODO dengan penanda `MULAI/AKHIR KODE DI SINI`
- Contoh expected output di `main()`
